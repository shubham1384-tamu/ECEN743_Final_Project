# ECEN 743 Final Project

## Description

This is the final project for ECEN 743.

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd ECEN743_Final_Project
   ```

2. Run the installation script to install all dependencies:
   ```bash
   ./install_dependencies.sh
   ```

   This script will:
   - Install Python packages from `requirements.txt`
   - Install `f110_gym` (F1TENTH Gym environment) from source
   - Handle any special dependencies required for TAMU HPRC

   **Note for TAMU HPRC users:** The script includes fixes for installing `f110_gym` in the HPRC environment. If you encounter issues, ensure you have internet access for Git cloning or use a local mirror.
